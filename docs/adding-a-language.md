# Adding a New Language to LaunchKit

This guide walks through adding support for a new language (e.g., Rust) to LaunchKit.

## Overview

You need to touch 4 areas:

1. **Detector** — teach LaunchKit to recognize the language in a project
2. **Docker generator** — create a Dockerfile template
3. **Config enum** — add the language to the `Lang` enum
4. **Tests** — verify everything works

## Step 1: Add the Language Enum

Edit `src/launchkit/core/config.py`:

```python
class Lang(str, Enum):
    PYTHON = "python"
    NODE = "node"
    GO = "go"
    RUST = "rust"      # ← add your language
```

## Step 2: Create a Detector

Create `src/launchkit/detectors/rust.py`:

```python
"""Rust stack detector."""

from __future__ import annotations

from pathlib import Path

from launchkit.detectors.base import BaseDetector, DetectedService


class RustDetector(BaseDetector):
    def detect(self, root: Path, name: str) -> DetectedService | None:
        cargo = root / "Cargo.toml"
        if not cargo.exists():
            return None

        framework = self._detect_framework(root)
        return DetectedService(
            name=name,
            lang="rust",
            framework=framework,
            port=8080,
        )

    def _detect_framework(self, root: Path) -> str | None:
        cargo = root / "Cargo.toml"
        content = cargo.read_text()
        if "actix-web" in content:
            return "actix"
        if "axum" in content:
            return "axum"
        if "rocket" in content:
            return "rocket"
        return None
```

### Register the Detector

In `src/launchkit/detectors/base.py`, add your detector to the `DETECTORS` list:

```python
from launchkit.detectors.rust import RustDetector  # or inline the class

DETECTORS: list[BaseDetector] = [
    PythonDetector(),
    NodeDetector(),
    GoDetector(),
    RustDetector(),    # ← add here
]
```

### Detection Principles

- **Be conservative.** A false negative (not detecting) is better than a false positive.
- **Check for the canonical project file** (`Cargo.toml`, `go.mod`, `package.json`, etc.).
- **Framework detection is optional.** Return `None` if unsure.
- **Port should match the language's convention** (e.g., 8080 for Go/Rust, 3000 for Node).

## Step 3: Create a Dockerfile Generator

Create `src/launchkit/generators/docker/rust.py`:

```python
"""Rust Dockerfile generator."""

from __future__ import annotations

from launchkit.core.config import ServiceConfig


def generate_rust_dockerfile(name: str, service: ServiceConfig) -> str:
    port = service.port or 8080

    lines = [
        "# Generated by LaunchKit",
        "",
        "FROM rust:1.77-slim AS builder",
        "WORKDIR /app",
        "COPY Cargo.toml Cargo.lock ./",
        "# Cache dependency builds",
        "RUN mkdir src && echo 'fn main() {}' > src/main.rs",
        "RUN cargo build --release && rm -rf src",
        "",
        "COPY . .",
        "RUN cargo build --release",
        "",
        "FROM debian:bookworm-slim AS runtime",
        "RUN apt-get update && apt-get install -y ca-certificates && rm -rf /var/lib/apt/lists/*",
        "COPY --from=builder /app/target/release/server /usr/local/bin/server",
        "",
        "RUN useradd --create-home appuser",
        "USER appuser",
        "",
        f"EXPOSE {port}",
        'CMD ["server"]',
        "",
    ]
    return "\\n".join(lines)
```

### Wire it into the Engine

In `src/launchkit/core/engine.py`, add the import and case:

```python
from launchkit.generators.docker.rust import generate_rust_dockerfile

def _build_dockerfile(name: str, service: ServiceConfig) -> str:
    lang = service.lang.value
    ...
    elif lang == "rust":
        return generate_rust_dockerfile(name, service)
```

## Step 4: Add Tests

Create `tests/unit/test_rust_detector.py`:

```python
from pathlib import Path
from launchkit.detectors.rust import RustDetector


def test_detects_rust(tmp_path: Path) -> None:
    (tmp_path / "Cargo.toml").write_text('[package]\nname = "myapp"\n')
    d = RustDetector()
    result = d.detect(tmp_path, "api")
    assert result is not None
    assert result.lang == "rust"


def test_detects_actix(tmp_path: Path) -> None:
    (tmp_path / "Cargo.toml").write_text('[dependencies]\nactix-web = "4"\n')
    d = RustDetector()
    result = d.detect(tmp_path, "api")
    assert result.framework == "actix"
```
